# 不同题型的UI样式区分

## 概述
现在根据题目类型显示不同的UI样式，提升用户体验和视觉区分度。

## 题型分类和UI样式

### 1. 单选题 (Single Choice) - 圆形单选按钮
**特点**：
- 每个选项前有一个圆形单选按钮
- 只能选择一个选项
- 选中时圆形内显示白色圆点
- 垂直排列，全宽显示

**样式细节**：
```
┌─────────────────────────────────┐
│ ○ 选项 A                        │
└─────────────────────────────────┘
┌─────────────────────────────────┐
│ ○ 选项 B                        │
└─────────────────────────────────┘
┌─────────────────────────────────┐
│ ● 选项 C (已选中)               │
└─────────────────────────────────┘
```

**代码实现**：
```typescript
// 单选圆形按钮
<div className="h-5 w-5 flex-shrink-0 rounded-full border-2 flex items-center justify-center">
  {isSelected && <div className="h-2 w-2 rounded-full bg-white" />}
</div>
```

---

### 2. 多选题 (Multiple Choice) - 方形复选框
**特点**：
- 每个选项前有一个方形复选框
- 可以选择多个选项
- 选中时显示白色对勾 (✓)
- 2列网格排列，更紧凑

**样式细节**：
```
┌──────────────────┐  ┌──────────────────┐
│ ☐ 选项 A        │  │ ☐ 选项 B        │
└──────────────────┘  └──────────────────┘
┌──────────────────┐  ┌──────────────────┐
│ ☑ 选项 C (已选) │  │ ☐ 选项 D        │
└──────────────────┘  └──────────────────┘
```

**代码实现**：
```typescript
// 多选方形复选框
<div className="h-5 w-5 flex-shrink-0 rounded-[4px] border-2 flex items-center justify-center">
  {isSelected && <Check className="h-3 w-3 text-white" />}
</div>
```

---

### 3. 填空题 (Fill in the Blank) - 文本输入框
**特点**：
- 显示为文本输入框样式
- 只能选择一个答案
- 垂直排列，全宽显示
- 边框更粗（border-2）

**样式细节**：
```
┌─────────────────────────────────┐
│ 答案选项 1                      │
└─────────────────────────────────┘
┌─────────────────────────────────┐
│ 答案选项 2                      │
└─────────────────────────────────┘
```

**代码实现**：
```typescript
// 填空题文本框
<button className="w-full rounded-[12px] border-2 px-4 py-3">
  {option}
</button>
```

## 颜色状态

所有题型都遵循相同的颜色状态：

| 状态 | 颜色 | 说明 |
|------|------|------|
| 未选中 | 白色 + 灰色边框 | 默认状态 |
| 已选中 | 蓝色背景 + 蓝色边框 | 用户选中 |
| 正确答案 | 绿色背景 + 绿色边框 | 提交后显示 |
| 错误答案 | 红色背景 + 红色边框 | 提交后显示 |

---

## 交互行为

### 单选题
- 点击选项：直接选中该选项，取消其他选项
- 提交后：禁用所有选项，显示正确/错误状态

### 多选题
- 点击选项：切换该选项的选中状态
- 可以同时选中多个选项
- 提交后：禁用所有选项，显示正确/错误状态

### 填空题
- 点击选项：选中该选项
- 提交后：禁用所有选项，显示正确/错误状态

## 布局对比

| 题型 | 排列方式 | 列数 | 按钮样式 |
|------|---------|------|---------|
| 单选题 | 垂直 | 1 | 圆形单选 |
| 多选题 | 网格 | 2 | 方形复选 |
| 填空题 | 垂直 | 1 | 文本框 |

## 用户体验改进

1. **视觉区分**：不同题型有明显的视觉差异
2. **交互直观**：UI样式暗示了交互方式（单选 vs 多选）
3. **空间利用**：多选题使用2列网格，更紧凑
4. **一致性**：颜色和状态反馈保持一致

## 测试场景

### 场景 1：单选题
1. 显示圆形单选按钮
2. 点击选项时，只有该选项被选中
3. 提交后，显示正确/错误状态

2. 点击多个选项时，都被选中
3. 提交后，显示正确/错误状态

### 场景 3：填空题
1. 显示文本框样式
2. 点击选项时，该选项被选中
3. 提交后，显示正确/错误状态
