# 学习广场功能设计说明（先文档后开发）

## 1. 目标
将 SparkLearn 的“学习广场”做成独立站点式页面（新标签页打开），视觉与交互参考你提供的论坛截图，并保持 SparkLearn 产品图标与风格体系一致。

## 2. 入口与导航策略

### 2.1 入口方式
- 主系统侧边栏点击“学习广场”时：
  - 新开标签页进入 `/plaza`
  - 不在主页面内嵌（非画中画、非 iframe）

### 2.2 页面层级
- `/plaza`：广场首页（帖子列表 + 就地发帖）
- `/plaza/[postId]`：帖子详情页

## 3. 页面布局（参考图映射）

### 3.1 顶部栏（全局）
- 左侧：`学习广场` 品牌名 + 当前版块名（如“使用交流”）
- 右侧：登录/用户入口（后续可接真实用户体系）
- 视觉：深色横栏，固定在顶部

### 3.2 左侧导航栏
- 一级：`首页`
- 一级：`分类版块`
- 二级板块列表：
  - 使用交流
  - 建议和反馈
  - 公告通知
  - 开源定制板块
- 每个板块显示帖子数量
- 个人中心区：
  - 个人信息
  - 消息中心

### 3.3 主内容区
- 版块信息卡（图标 + 板块名 + 简述）
- 帖子列表工具条：
  - 帖子总数
  - 筛选
  - 排序（最新回复/最新发布/热门）
  - 搜索入口
- 帖子列表项：
  - 标题（可显示“置顶”标识）
  - 作者信息
  - 最后回复时间
  - 评论数

### 3.4 右下悬浮操作
- 悬浮“+”按钮：触发就地发帖面板（抽屉/弹层）
- 不跳转新页面

## 4. 功能清单（V1）

### 4.1 帖子浏览
- 支持按板块查看帖子
- 支持分页/滚动加载（二选一，默认分页）
- 支持帖子详情跳转

### 4.2 帖子搜索（模糊）
- 搜索范围：`标题 + 正文 + 标签`
- 搜索方式：模糊匹配（LIKE）
- 支持 Enter 提交与按钮提交

### 4.3 帖子筛选与排序
- 筛选：按板块、是否置顶
- 排序：
  - 最新回复
  - 最新发布
  - 热门（评论/点赞/收藏加权）

### 4.4 发帖与资料共享（就地）
- 触发方式：点击右下角 `+`
- 展示方式：当前 `/plaza` 页面内弹层/抽屉
- 发帖字段：标题、正文、标签、附件
- 附件支持：pdf/doc/docx/ppt/pptx/xls/xlsx/zip/png/jpg/jpeg
- 附件下载：详情页可下载

### 4.5 互动能力
- 点赞（toggle）
- 收藏（toggle）
- 评论（新增）

## 5. 图标系统规范

- 图标来源：统一使用项目当前图标体系（`lucide-react` + 现有 UI 组件）
- 禁止引入与主产品不一致的第三方图标包
- 图标尺寸建议：
  - 顶部/导航：16~20
  - 列表操作：14~16

## 6. 接口与前后端映射

### 6.1 已有接口（继续沿用）
- `GET /api/forum/posts`
- `POST /api/forum/posts`
- `GET /api/forum/posts/{post_id}`
- `POST /api/forum/posts/{post_id}/comments`
- `POST /api/forum/posts/{post_id}/like`
- `POST /api/forum/posts/{post_id}/favorite`
- `POST /api/forum/posts/{post_id}/attachments`
- `GET /api/forum/attachments/{attachment_id}/download`

### 6.2 前端页面调用关系
- `/plaza`：调用 `getForumPosts({ tab, q, tag, page })`
- `/plaza` 内发帖弹层：调用 `createForumPost + uploadForumAttachments`
- `/plaza/[postId]`：调用 `getForumPostDetail + getForumComments + createForumComment + toggleLike/Favorite`

## 7. 状态与异常处理

- 空列表：显示“暂无帖子”
- 搜索无结果：显示“未找到相关帖子”
- 网络失败：显示错误提示 + 重试按钮
- 提交中：按钮禁用，显示 loading 文案

## 8. 验收标准（开发前确认）

1. 从主系统点击学习广场，会新开标签页进入 `/plaza`
2. `/plaza` 具备“顶部栏 + 左侧导航 + 主内容区”三段式布局
3. 列表支持模糊搜索（标题/正文/标签）
4. 点击右下角 `+` 可直接在当前页完成发帖（不跳转）
5. 可进入详情页进行评论、点赞、收藏
6. 图标全部使用产品统一图标体系

## 9. 开发顺序（确认后执行）

1. 结构层：`/plaza` 独立 layout + 顶部栏 + 左侧栏
2. 列表层：板块头、工具条、帖子列表
3. 交互层：搜索/筛选/排序 + 右下角加号发帖弹层
4. 业务层：详情、评论、点赞、收藏、附件
5. 联调与测试：接口回归 + 页面验收
