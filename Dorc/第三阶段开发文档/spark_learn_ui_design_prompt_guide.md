# SparkLearn 蓝色大厂风 UI 设计提示词规范

> 适用于 SparkLearn / 学而思个性化学习平台的页面重构、AI 生图、前端高保真还原、产品原型设计。  
> 核心目标：在「AI 教育 SaaS」基础上，融合「浅蓝大厂科技风」「玻璃拟态」「轻游戏化」「可爱学习伙伴」四类视觉语言，形成统一、干净、高级、可落地的产品界面风格。

---

## 1. 总体设计定位

请设计一套符合 SparkLearn 学而思个性化学习平台的 UI 界面，整体风格为浅蓝色大厂科技风，融合 AI 教育产品、SaaS 工作台、学习助手、游戏化学习伙伴系统的视觉语言。页面需要体现智能、可信、清爽、温暖、陪伴感和轻游戏化体验。

整体不要做成普通后台管理系统，也不要做成低幼儿童游戏界面。它应该像一个成熟大厂推出的 AI 学习平台，既有企业级产品的专业感，也有学习伙伴系统的亲和力和趣味性。

关键词：浅蓝科技风、AI 教育 SaaS、玻璃拟态、大厂官网感、强留白、柔和渐变、学习伙伴、宠物养成、轻游戏化、精致圆角、低饱和背景、高亮蓝按钮、温暖陪伴感。


## 4. 背景设计规范

背景是整个项目风格的关键。需要使用浅蓝到白色的柔和渐变，搭配抽象科技光线、弧形轨道、轻微粒子、柔和光晕，形成通透的科技空间感。

### 背景提示词

```text
设计一个浅蓝白色科技感背景，整体明亮、通透、干净。背景使用白色到冰蓝色的柔和渐变，左右两侧加入对称的浅蓝光线纹理和弧形轨道线，中间保持大面积留白，方便放置主要内容卡片。光线需要柔和、低透明度，不要过度抢占内容层级。加入少量微弱发光点、细线、抽象科技波纹和轻微空间透视感。整体要像 AI 教育 SaaS 产品的高级背景，而不是游戏海报或赛博朋克背景。
```

### 对称背景提示词

```text
请基于浅蓝白色科技背景做左右对称设计，左右两侧都有柔和的光线弧线、淡蓝渐变和轻微发光节点，中间区域保持干净留白，用于承载玻璃拟态主卡片。整体要高级、克制、通透，左右不能一边很空一边很重。
```

### 背景要求

- 中心留白，避免影响阅读。
- 光线分布要平衡，左右不能明显失衡。
- 弧线和粒子只做氛围，不做主视觉。
- 页面右侧可以有更强的科技光束，左侧也要有轻微纹理呼应。
- 背景透明度要低，不能压过卡片和文字。
- 适合所有页面复用，但每页可以微调光线位置。

## 5. 卡片设计规范

卡片是页面主要容器，统一使用白色或半透明白色玻璃拟态效果。

### 基础卡片样式

```text
卡片使用白色或半透明白色背景，圆角 20px 到 28px，边框为极浅蓝灰色，带柔和阴影。卡片内部留白充足，内容对齐严谨。整体像悬浮在浅蓝背景上的玻璃卡片，不能显得厚重或廉价。
```

### 推荐 CSS 风格

```css
.card {
  background: rgba(255, 255, 255, 0.86);
  border: 1px solid rgba(210, 225, 245, 0.85);
  border-radius: 24px;
  box-shadow: 0 18px 50px rgba(42, 91, 160, 0.08);
  backdrop-filter: blur(18px);
}
```

### 卡片层级

```text
一级大卡片：用于页面主体、流程容器、主要工作区。
二级内容卡片：用于功能模块、宠物属性、任务列表、资源推荐。
三级小卡片：用于数据指标、标签、按钮组、快捷入口。
列表项卡片：用于任务、资源、风格选择、学习时间选择。
```

### 卡片禁忌

- 不要每个内容都做成同样大小的卡片，容易割裂。
- 不要使用厚重阴影。
- 不要使用黑灰边框。
- 不要堆太多彩色卡片。
- 不要让卡片像移动端小程序随意拼贴。
- 信息密度要控制，核心操作突出。

## 6. 图标设计规范

推荐使用线性图标库，再套彩色圆角底块。不要直接混用大量不同风格的彩色图标。

### 推荐图标库

```text
Lucide Icons：最推荐，适合现代 SaaS 和 AI 产品
Tabler Icons：适合学习平台、数据面板、后台页面
Phosphor Icons：适合柔和、圆润、轻游戏化界面
Heroicons：适合基础导航、按钮、操作类图标
Iconify：适合统一搜索多个图标集，但需要注意统一风格
```

### 图标样式

```text
线性图标 + 彩色浅底圆角块
图标大小 18px-28px
底块大小 36px-56px
圆角 12px-18px
线宽统一
颜色低饱和
```

### 图标底块示例

```text
蓝色底块 + 文档 / 学习 / 搜索图标
紫色底块 + AI / 技能 / 魔法 / 分析图标
绿色底块 + 完成 / 成长 / 资源图标
橙色底块 + 任务 / 奖励 / 能量图标
粉色底块 + 亲密度 / 陪伴 / 爱心图标
```

## 7. 字体与排版规范

整体字体要清晰、现代、干净，避免花哨字体。

### 字体建议

```text
中文：PingFang SC / Microsoft YaHei / 思源黑体
英文：Inter / SF Pro Display / system-ui
数字：Inter / DIN 风格数字
```

### 字体层级

```text
页面主标题：28px-36px，字重 700-800，深蓝色
模块标题：18px-22px，字重 600-700，深蓝色
卡片标题：15px-18px，字重 600
正文说明：13px-15px，灰蓝色
辅助文字：12px-13px，浅灰蓝色
数字指标：24px-32px，字重 700
```

### 排版要求

- 标题要有力量感，但不要过大。
- 正文行距舒适，避免挤压。
- 主标题和副标题形成清晰层级。
- 页面顶部保留面包屑和页面说明。
- 卡片内文字左对齐为主，重要流程页可居中。
- 不要出现杂乱加粗、彩色高亮过多的问题。

## 8. 页面通用布局

### 基础布局结构

```text
左侧固定导航栏 + 顶部面包屑区域 + 主内容区 + 右侧背景科技光线
```

### 左侧导航栏

```text
左侧导航栏保持浅色背景，顶部放 SparkLearn Logo 和产品名，下方按功能分组展示导航。当前页面使用浅蓝选中底色和蓝色图标。底部显示用户信息卡片和退出入口。
```

### 主内容区

```text
主内容区不铺满屏幕，保持舒适宽度。内容容器居中或偏左，根据页面类型决定。背景光线主要在右侧和两侧边缘，中间保持清爽。
```

### 顶部区域

```text
顶部显示面包屑，例如 SparkLearn / 学习伙伴。页面标题位于主区域左上方，下面放一句说明文字。右上角可显示金币、等级、头像、通知等状态信息。
```
