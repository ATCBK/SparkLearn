# 前端背景图/装饰图改造方案

## 一、概述

当前前端页面大部分使用纯色/渐变背景，缺少视觉层次感。本方案为各页面预留背景图和装饰图位置，后续只需提供图片文件即可快速贴入。

---

## 二、图片存放规则

| 项目 | 说明 |
|------|------|
| 源文件目录 | `D:\Project_building\SparkLearn\UI_image\` |
| 前端引用目录 | `frontend/public/ui-images/` |
| 引用方式 | `<img src="/ui-images/xxx.png" />` 或 CSS `background-image: url('/ui-images/xxx.png')` |
| 命名规范 | `{页面名}-{位置}.png`，如 `home-hero.png`、`profile-card.png` |

> 图片需要从 `UI_image/` 复制到 `frontend/public/ui-images/` 才能被前端访问。

## 四、代码改造步骤

### Step 1：修改 ProtoCard 组件支持 style 透传

```tsx
// frontend/src/components/proto/index.tsx
export function ProtoCard({
  children,
  className,
  style,  // 新增
}: {
  children: React.ReactNode
  className?: string
  style?: React.CSSProperties  // 新增
}) {
  return (
    <section style={style} className={cn('rounded-[12px] border border-line bg-white p-5 shadow-md', className)}>
      {children}
    </section>
  )
}
```

### Step 2：改造学习工作台 Hero 卡片

```tsx
<ProtoCard
  className="relative overflow-hidden p-[22px] border-0"
  style={{ background: 'linear-gradient(135deg, #1e40af 0%, #3b82f6 50%, #60a5fa 100%)' }}
>
  {/* 右侧装饰图 */}
  <img
    src="/ui-images/home-hero.png"
    alt=""
    className="pointer-events-none absolute right-4 top-1/2 -translate-y-1/2 h-[80%] w-auto max-w-[300px] object-contain opacity-90 max-[900px]:hidden"
  />
  <div className="relative z-10 max-w-[680px] space-y-3.5">
    {/* 文字内容，颜色改为白色 */}
  </div>
</ProtoCard>
```

### Step 3：为其他页面预留 PageHead 背景 slot

在 `PageHead` 组件中新增可选的 `backgroundImage` 属性：

```tsx
export function PageHead({
  eyebrow, title, description, chips,
  backgroundImage,  // 新增：可选背景图路径
}: {
  // ...existing props
  backgroundImage?: string
}) {
  return (
    <header className="relative mb-5 overflow-hidden rounded-[12px] border-b border-line pb-4">
      {backgroundImage && (
        <img
          src={backgroundImage}
          alt=""
          className="pointer-events-none absolute inset-0 h-full w-full object-cover opacity-10"
        />
      )}
      <div className="relative z-10">
        {/* existing content */}
      </div>
    </header>
  )
}
```

## 七、注意事项

- 图片建议使用 PNG 透明背景（装饰图）或 JPG（大背景）
- 装饰图不要太大，控制在 200KB 以内保证加载速度
- 所有图片都是纯视觉装饰，不影响功能逻辑
- 如果某个页面暂时没有图片，代码会优雅降级（不显示图片，保持原样）
