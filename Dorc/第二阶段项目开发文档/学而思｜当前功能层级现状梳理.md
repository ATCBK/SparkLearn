# 学而思｜当前功能层级现状梳理

## 1. 文档目的

本文档用于盘点 SparkLearn 当前已有页面、显性功能、能力树、数据结构和赛题对应关系，是第二阶段功能树重构前的现状基线。本文档只回答“当前项目已经有什么”，不回答“最终应该改成什么”。

本文档聚焦四个问题：

第一，当前前端页面和主应用入口是如何组织的。

第二，学习画像、学习路径、资源、练习、智能辅导和学习报告等能力当前分别落在哪些页面中。

第三，当前数据层已经有哪些 SQLite 表、JSON 文件和 JSONL 事件记录。

第四，已有功能与赛题要求之间目前如何对应。

本文档是现状盘点文档，不是目标功能树，也不是开发清单。文中记录的当前页面树不能直接作为第二阶段目标结构使用。后续未实现或待落地内容需查看功能层级改造方案、资源生成与资源库页面更新设计，以及具体代码实现记录。

## 5. 当前数据树状图

```text
数据层
├─ SQLite
│  ├─ students：学生基础信息
│  ├─ profiles：学习画像
│  ├─ mastery_records：知识点掌握度
│  ├─ contribution_days：学习打卡
│  ├─ tutor_roles：导师角色
│  ├─ tutor_conversations：Tutor 会话
│  ├─ tutor_messages：Tutor 消息
│  └─ tutor_files：Tutor 上传文件
│
└─ JSON / JSONL
   ├─ profile_snapshot.json：画像快照
   ├─ learning_events.jsonl：学习行为事件
   ├─ latest_quiz_set.json：最近题目批次
   ├─ quiz_records.json：练习记录
   ├─ quiz_favorites.json：收藏题
   ├─ resources_index.json：资源索引
   ├─ resource_usage.jsonl：资源使用记录
   ├─ tutor_history.json：Tutor 历史
   ├─ uploads/：上传文件
   └─ ppt_decks/：PPT 生成产物
```
