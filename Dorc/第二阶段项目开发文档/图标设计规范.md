# SparkLearn 图标设计规范

## 一、设计原则

- **统一图标库**：全项目使用 Lucide Icons（`lucide-react`），不混用其他图标库
- **彩色感来源**：图标本身保持线性（stroke），彩色感交给外层圆角底块和浅色背景
- **视觉层次**：通过底块尺寸区分信息层级（指标 > 列表项 > 标题装饰）

## 三、尺寸规范

### 3.1 统计指标图标（大圆形）

用于页面顶部的数据统计栏，每个指标左侧配一个大圆形图标。

```tsx
<div className="grid h-11 w-11 shrink-0 place-items-center rounded-full bg-[#eff6ff] text-[#2563eb]">
  <PieChart className="h-5 w-5" />
</div>
```

| 属性 | 值 |
|------|-----|
| 容器尺寸 | `h-11 w-11`（44px） |
| 圆角 | `rounded-full` |
| 图标尺寸 | `h-5 w-5`（20px） |

### 3.2 列表项图标（中方块）

用于任务列表、资源列表、错题列表等每一行的左侧。

```tsx
<div className="grid h-9 w-9 shrink-0 place-items-center rounded-xl bg-[#f3efff] text-[#7c3aed]">
  <FileText className="h-4 w-4" />
</div>
```

| 属性 | 值 |
|------|-----|
| 容器尺寸 | `h-9 w-9`（36px） |
| 圆角 | `rounded-xl` |
| 图标尺寸 | `h-4 w-4`（16px） |

### 3.3 标题装饰图标（小方块）

用于卡片/区块标题前的装饰性图标。

```tsx
<div className="grid h-7 w-7 place-items-center rounded-lg bg-[#ecfdf5] text-[#059669]">
  <CalendarDays className="h-4 w-4" />
</div>
```

| 属性 | 值 |
|------|-----|
| 容器尺寸 | `h-7 w-7`（28px） |
| 圆角 | `rounded-lg` |
| 图标尺寸 | `h-4 w-4`（16px） |

## 四、IconBlock 组件

项目中封装了统一的 `IconBlock` 组件，位于 `frontend/src/components/ui/IconBlock.tsx`。

```tsx
import { IconBlock } from '@/components/ui/IconBlock'
import { FileText, BarChart3, Database } from 'lucide-react'

// 基本用法
<IconBlock icon={<FileText />} color="blue" />
<IconBlock icon={<BarChart3 />} color="purple" size="lg" />
<IconBlock icon={<Database />} color="teal" size="sm" />
```

### Props

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `icon` | `React.ReactNode` | — | Lucide 图标组件 |
| `color` | `'blue' \| 'purple' \| 'green' \| 'orange' \| 'red' \| 'teal' \| 'pink'` | `'blue'` | 配色 |
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` | 尺寸 |
| `className` | `string` | — | 额外样式 |

### 尺寸映射

| size | 容器 | 图标 |
|------|------|------|
| `sm` | `h-8 w-8 rounded-lg` | `h-4 w-4` |
| `md` | `h-10 w-10 rounded-xl` | `h-5 w-5` |
| `lg` | `h-12 w-12 rounded-xl` | `h-6 w-6` |

## 七、新增页面/组件时的图标选择指南

1. **确定功能类别** → 选择对应颜色
2. **确定信息层级** → 选择对应尺寸（统计/列表/标题）
3. **从 Lucide 选择语义最接近的图标**
4. **使用 `IconBlock` 组件或手写底块样式**

## 八、注意事项

- 不要在同一个页面使用超过 4 种颜色的图标底块
- 同一列表中的图标底块应保持相同尺寸
- 图标底块不应有 hover/交互效果（它是装饰性的）
- 如果图标底块内的图标需要交互（如勾选），单独处理交互逻辑
- 保持 `shrink-0` 防止图标在 flex 布局中被压缩
