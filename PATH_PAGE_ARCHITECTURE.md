## 📊 数据流

```
┌─────────────────────────────────────────────────────────────┐
│                    数据定义 (顶部)                           │
│                                                               │
│  PHASES: Phase[]                                             │
│  ├── Phase 1 (补弱)                                          │
│  │   └── nodes: PathNode[] (3 个)                            │
│  ├── Phase 2 (达标)                                          │
│  │   └── nodes: PathNode[] (4 个)                            │
│  └── Phase 3 (目标)                                          │
│      └── nodes: PathNode[] (3 个)                            │
│                                                               │
│  SUGGESTIONS: Suggestion[]                                   │
│  ├── Suggestion 1                                            │
│  ├── Suggestion 2                                            │
│  └── Suggestion 3                                            │
│                                                               │
│  RESOURCES: Resource[]                                       │
│  ├── Resource 1                                              │
│  ├── Resource 2                                              │
│  └── Resource 3                                              │
└─────────────────────────────────────────────────────────────┘
         │
         ▼
┌─────────────────────────────────────────────────────────────┐
│                    状态管理 (useState)                       │
│                                                               │
│  targetInput: string                                         │
│  selectedNodeId: number | null                               │
└─────────────────────────────────────────────────────────────┘
         │
         ▼
┌─────────────────────────────────────────────────────────────┐
│                    组件渲染                                   │
│                                                               │
│  PathCircuitCard                                             │
│  ├── 接收: targetInput, setTargetInput                       │
│  ├── 接收: selectedNodeId, setSelectedNodeId                 │
│  └── 渲染: PathCanvas + 图例 + 按钮                          │
│                                                               │
│  PathCanvas                                                  │
│  ├── 接收: selectedNodeId, setSelectedNodeId                 │
│  └── 渲染: PhaseRow × 3 + TurnLine × 2 + 结束标记            │
│                                                               │
│  PhaseRow                                                    │
│  ├── 接收: phase, top, selectedNodeId, setSelectedNodeId     │
│  └── 渲染: 阶段标识 + PathNode × N + 连接线                  │
│                                                               │
│  PathNode                                                    │
│  ├── 接收: node, isSelected, onClick                         │
│  └── 渲染: 节点卡片 + 状态标签 + 完成标记                    │
│                                                               │
│  SuggestionPanel                                             │
│  ├── 接收: selectedNodeId                                    │
│  └── 渲染: 建议卡片 + 资源卡片 + 操作链接                    │

```

## 🔄 交互流程

### 1. 页面初始化
```
1. 组件挂载
2. 初始化状态:
   - targetInput = '我想自己写一个成绩统计程序'
   - selectedNodeId = 4 (当前推荐节点)
3. 渲染路径图
4. 第 4 个节点高亮
5. 右侧显示建议
```

### 2. 点击节点
```
1. 用户点击节点
2. 触发 onClick 事件
3. 调用 setSelectedNodeId(nodeId)
4. 节点高亮更新
5. 右侧面板可更新内容
```

### 3. 输入目标
```
1. 用户在输入框输入
2. 触发 onChange 事件
3. 调用 setTargetInput(value)
4. 输入框值更新
```

### 4. 生成路径
```
1. 用户点击"生成路径"按钮
2. 调用 API: generatePath(targetInput)
3. 获取新的路径数据
4. 更新 PHASES 数据
5. 重新渲染路径图
```

### 5. 点击操作按钮
```
学习资源:
  1. 用户点击按钮
  2. 调用 router.push('/resources')
  3. 导航到资源页面

进入练习:
  1. 用户点击按钮
  2. 调用 router.push('/practice')
  3. 导航到练习页面

生成资源:
  1. 用户点击按钮
  2. 调用 router.push('/generate')
  3. 导航到生成页面
```

## 🔌 API 集成点

### 后端 API 调用
```typescript
// 获取路径数据
GET /api/path/data
Response: {
  phases: Phase[]
  suggestions: Suggestion[]
  resources: Resource[]
}

// 生成新路径
POST /api/path/generate
Request: { target: string }
Response: { phases: Phase[] }

// 获取节点详情
GET /api/path/nodes/:nodeId
Response: { node: PathNode, details: {...} }

// 更新节点状态
PUT /api/path/nodes/:nodeId
Request: { status: 'completed' | 'current' | 'next' | 'locked' }
Response: { success: boolean }
```

## 🚀 性能优化

### 当前优化
- 使用 Tailwind CSS (预编译)
- 组件化设计 (减少重排)
- 状态最小化 (只有 2 个状态)

### 后续优化建议
```
1. React.memo 优化节点组件
   - 避免不必要的重新渲染

2. useCallback 缓存事件处理
   - 避免函数重新创建

3. useMemo 缓存计算结果
   - 避免重复计算

4. 虚拟滚动
   - 如果节点数量很多

5. 代码分割
   - 按需加载组件
```
