# 按钮位置调整

## 功能描述
将"错题本"和"收藏题目"按钮从题目卡片移动到页面顶部的"练习评测"标题旁边。

## 变更内容

### 之前的位置
- 位置：题目卡片下方，"提交判题"按钮旁边
- 显示：在每道题目下方都显示

### 现在的位置
- 位置：页面顶部，"练习评测"标题右侧（PageHead 的 actions 区域）
- 显示：页面加载时就显示，始终可见

## 代码变更

### PageHead 配置
```typescript
<PageHead
  eyebrow="练习评测 / 判题、解析、错题"
  title="练习评测"
  description="完成针对性练习后，系统会回写画像、错题本和学习路径。"
  actions={
    <div className="flex gap-2">
      <ProtoButton href="/practice/mistakes" variant="secondary">
        错题本
      </ProtoButton>
      <ProtoButton href="/practice/favorites" variant="secondary">
        收藏题目
      </ProtoButton>
    </div>
  }
  chips={[
    { value: `${questions.length} 题`, label: '本次练习' },
    { value: '80%', label: '达标线' },
    { value: `${accuracy}%`, label: '当前正确率' },
  ]}
/>
```

### 移除的代码
从题目卡片的按钮组中移除了"收藏题目"按钮：
```typescript
// 之前
<ProtoButton variant="secondary" onClick={() => api.setQuizFavorite(currentQuestion.id, true)}>
  收藏题目
</ProtoButton>

// 现在
// 已移除
```

## 视觉效果

### 之前
```
┌─────────────────────────────────────────┐
│ 练习评测                                │
│ 完成针对性练习后，系统会回写画像...    │
└─────────────────────────────────────────┘

┌─────────────────────────────────────────┐
│ 第 1 题：题目内容                       │
│ ○ 选项 A                               │
│ ○ 选项 B                               │
│ [提交判题] [收藏题目]                   │  ← 按钮在这里
└─────────────────────────────────────────┘
```

### 现在
```
┌─────────────────────────────────────────┐
│ 练习评测              [错题本] [收藏题目]│  ← 按钮在这里
│ 完成针对性练习后，系统会回写画像...    │
└─────────────────────────────────────────┘

┌─────────────────────────────────────────┐
│ 第 1 题：题目内容                       │
│ ○ 选项 A                               │
│ ○ 选项 B                               │
│ [提交判题]                              │
└─────────────────────────────────────────┘
```

## 用户体验改进

1. **更清晰的布局**：按钮不再与题目卡片混在一起
2. **始终可见**：用户随时可以访问错题本和收藏题目
3. **减少视觉混乱**：题目卡片只显示必要的操作按钮
4. **更好的信息层级**：全局操作（错题本、收藏）与题目操作（提交）分离
