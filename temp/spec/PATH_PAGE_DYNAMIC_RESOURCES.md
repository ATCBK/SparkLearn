# 个性化路径动态资源推送功能

## 功能描述
更新学习路径页面，使其能够根据用户点击的节点动态生成对应的路径建议和资源推送。

## 核心变更

### 1. 节点显示优化

#### 之前
- 节点编号显示为大圆形（主要展示）
- 简化目标显示在下方
- 状态标签显示在底部

#### 现在
- **简化目标** - 主要展示（大字体）
- **节点编号** - 次要展示（小字体 "第 X 步"）
- **箭头图标** - 表示进度

### 2. 动态资源生成

#### 功能流程
```
用户点击节点
    ↓
触发 handleNodeClick() 函数
    ↓
根据节点ID获取对应的目标
    ↓
生成对应的建议和资源
    ↓
实时更新右侧面板
```

#### 生成规则
- **路径建议**：根据节点目标生成3条建议
  - 先学习 [目标]
  - 完成 [目标] 相关练习
  - 掌握 [目标] 的核心概念

- **资源推送**：根据节点目标生成3个资源
  - [目标]精讲讲义
  - [目标]补弱题练
  - [目标]项目案例

### 3. 代码实现

#### 主要函数

```typescript
const handleNodeClick = (nodeId: number) => {
  setSelectedNodeId(nodeId)
  
  // 获取节点对应的目标
  const goal = nodeGoalMap[nodeId] || '学习任务'
  
  // 生成建议
  const newSuggestions = [
    { id: 1, text: `先学习 ${goal}`, desc: '根据当前节点自动推荐的学习路径。' },
    { id: 2, text: `完成 ${goal} 相关练习`, desc: '根据当前节点自动推荐的练习任务。' },
    { id: 3, text: `掌握 ${goal} 的核心概念`, desc: '根据当前节点自动推荐的学习重点。' },
  ]
  
  // 生成资源
  const newResources = [
    { id: 1, title: `${goal}精讲讲义`, tag: '优先学习' },
    { id: 2, title: `${goal}补弱题练`, tag: '5题' },
    { id: 3, title: `${goal}项目案例`, tag: '待复习' },
  ]
  
  setGeneratedSuggestions(newSuggestions)
  setGeneratedResources(newResources)
}
```

| 2 | 掌握作用域 | 补弱 |
| 3 | 补弱练习 | 补弱 |
| 4 | 阶段完成 | 补弱 |
| 5 | 回顾函数 | 达标 |

| 12 | 项目完成 | 目标 |

### 用户操作
1. 用户查看学习路径
2. 点击某个节点（例如节点5）
3. 右侧面板实时更新
4. 显示该节点对应的建议和资源

1. 识别点击的节点ID
2. 查询节点对应的目标
3. 生成个性化建议
4. 生成对应资源列表
5. 更新UI显示
