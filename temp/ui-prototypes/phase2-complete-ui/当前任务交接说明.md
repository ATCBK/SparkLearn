## 2. 当前已完成（以代码现状为准）

### 2.1 全局框架

1. 已统一为产品化 SaaS 骨架：左侧导航 + 顶部工具栏 + 主内容区。  
2. 顶部已去掉全局搜索、通知，仅保留面包屑与用户信息。  
3. 风格已统一为浅色、低阴影、统一圆角和边框。

### 2.2 路由/页面结构

当前主要页面（含子页）：

- `dashboard` 学习工作台
- `profile` 学习画像
- `path` 个性化路径
- `generate` 资源中心
- `knowledge` 我的资料库
- `practice` 练习评测
- `mistakes` 错题本（练习子页）
- `favorites` 收藏题（练习子页）
- `report` 学习报告
- `loop` 学习闭环

说明：`mistakes/favorites` 高亮仍归属 `practice`；`knowledge` 高亮归属 `generate/resources` 组。

### 2.3 资源中心（核心改造）

1. 右上内容已改为 **6 步流程节点**（替代原先零散信息）。  
2. 中部改为更完整的流程工作区（非纯展示卡片）。  
3. “参考我的资料”已改为 **按钮 + 弹窗** 形式：  
   - 打开：`#openSourceModal`  
   - 弹窗：`#sourceModal`  
4. 弹窗确认后会把选中资料写回当前生成上下文。

### 2.4 我的资料库（知识库上传管理）

1. 已有资料管理页 `#knowledge`，不是独立新站点。  
2. 支持“上传资料用于问答/生成/复习建议”的产品化表达。  
3. 与资源生成流程有联动，可作为生成时参考来源。  
4. 已补“返回路径闭环”（从资料库可回到资源中心流程）。

### 2.5 练习评测（含错题/收藏）

1. 练习页已补齐 **错题本** 与 **收藏题** 入口（标题旁可跳转）。  
2. 已有独立详情页 `#mistakes`、`#favorites`。  
3. 右侧出题区支持“指定生成题目数量”（步进器 + 生成按钮）。  
4. 右下角重复“错题本”入口已去除，页面布局做过满屏收敛。

### 2.6 AI 学习助手（悬浮）

1. AI 助手改为右下可拖拽悬浮组件，不占主布局。  
2. 支持展开/缩小两态。  
3. **缩小态也支持拖动**（已按最新要求实现）。  
4. 页面内触发 AI 时会自动从缩小态展开。

### 2.7 工作台交互

1. “学习闭环记录”已改为“每日资源推荐”。  
2. 今日任务支持用户创建、勾选完成、删除（原型级交互）。
