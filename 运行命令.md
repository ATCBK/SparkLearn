# SparkLearn 运行命令

## 标准启动

### 后端
```bash
cd D:\Project_building\SparkLearn\backend
python -m uvicorn app.main:app --host 127.0.0.1 --port 8000 --reload
```

### 前端
```bash
cd D:\Project_building\SparkLearn\frontend
npm run dev
```

## 访问地址

| 功能 | 地址 |
|------|------|
| 学生端主应用 | http://localhost:3000 |
| 教师智能大屏 | http://localhost:3000/screen/index.html |
| 后端 API 文档 | http://127.0.0.1:8000/docs |
| 健康检查 | http://127.0.0.1:8000/health |

## 教师大屏说明

大屏位于 `frontend/public/screen/index.html`，通过 Next.js 静态服务访问。

- **后端已启动**：自动拉取真实班级数据（9名学生，含真实用户+8名模拟学生）
- **后端未启动**：自动降级为演示数据，大屏仍可正常展示所有图表

从学生端进入大屏：侧边栏 → 教师工具 → 教师大屏（新标签页打开）
