# SparkLearn 运行命令

## 标准启动

### 后端
```bash
cd D:\Project_building\SparkLearn\backend
python -m uvicorn app.main:app --host 127.0.0.1 --port 8000 --reload
```

### 学习宠物 + Nanobot 自动联动

当后端启动时，可自动拉起本地 `nanobot` API 服务，无需再单独开一个终端。

需要在项目根目录 `.env` 中至少配置以下项：

```bash
NANOBOT_PET_ENABLED=true
NANOBOT_AUTO_START=true
NANOBOT_API_BASE_URL=http://127.0.0.1:8900
NANOBOT_PROJECT_DIR=D:\Project_building\SparkLearn\nanobot-main
```

可选项：

```bash
NANOBOT_CONFIG_PATH=C:\Users\你的用户名\.nanobot\config.json
NANOBOT_WORKSPACE=D:\Project_building\SparkLearn\nanobot-main\.workspace
```

说明：

- 后端启动时会执行 `python -m nanobot serve`
- 如果 `8900` 端口上已经有健康的 `nanobot`，后端不会重复拉起
- 如果端口被占用但健康检查失败，后端启动会报错，避免静默冲突
- `nanobot` 日志会写入 `backend/data/logs/nanobot.stdout.log` 和 `backend/data/logs/nanobot.stderr.log`

### 前端


## 教师大屏说明

大屏位于 `frontend/public/screen/index.html`，通过 Next.js 静态服务访问。

- **后端已启动**：自动拉取真实班级数据（9名学生，含真实用户+8名模拟学生）
- **后端未启动**：自动降级为演示数据，大屏仍可正常展示所有图表

从学生端进入大屏：侧边栏 → 教师工具 → 教师大屏（新标签页打开）
